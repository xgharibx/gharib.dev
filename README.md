## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ 
- pnpm, npm, or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/xgharibx/gharib.dev.git
cd gharib.dev

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Initialize the database
npm run db:generate
npm run db:push

# Start the development server
npm run dev
```
